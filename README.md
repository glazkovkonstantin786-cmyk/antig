## 📱 КАК УСТАНОВИТЬ НА ТЕЛЕФОН (Android)

### Способ 1 — Через GitHub Actions (APK, рекомендуется)
1. Перейди в репозиторий на GitHub
2. Вкладка **Actions** → запусти **"Build Android APK"**
3. После завершения (зелёная галочка) → скачай `antigravity-mobile-apk`
4. Включи на телефоне: **Настройки → Безопасность → Установка из неизвестных источников**
5. Установи `.apk` файл

### Способ 2 — Через браузер (PWA, мгновенно)
1. Открой **Chrome** на телефоне
2. Перейди на страницу GitHub Pages проекта
3. Нажми **"Добавить на главный экран"** (три точки → Добавить на главный экран)
4. Готово — иконка появится как у обычного приложения

## 💻 КАК ЗАПУСТИТЬ НА НОУТБУКЕ

### Способ 1 — Просто открыть файл (без установки)
Просто дважды кликни по файлу `index.html` в папке:
```
c:\Antigravity\AGENTS\AntigravityMobile\index.html
```

### Способ 2 — Локальный сервер (для всех функций)
```bash
# В терминале из папки проекта:
python -m http.server 3000
# Потом открой: http://localhost:3000
```

## 🧠 КАК ЗАПУСТИТЬ GravityBrain (своя нейросеть)

```bash
# Шаг 1: Установить зависимости (один раз)
pip install -r brain/requirements.txt

# Шаг 2: Обучить нейросеть (5-15 минут на CPU)
python brain/train_brain.py --epochs 5

# Шаг 3: Запустить сервер
python brain/server.py

# Шаг 4: Открыть index.html, выбрать "⚡ GravityBrain v1.0"
```

Добавь свои тексты в папку `brain/data/` перед обучением, чтобы нейросеть стала экспертом именно в твоих темах.
